# ThemeAtlas Icons

Primary icon files for app branding:

- `themeatlas-icon.svg`: Full app icon (rounded square, atlas ring + compass needle)
- `themeatlas-mark.svg`: Minimal mark for badges and smaller placements

Suggested exports:

- 1024x1024 for app stores/release assets
- 512x512 for desktop launchers
- 256x256, 128x128, 64x64 for Linux icon themes

If you need PNG exports, convert from the SVG master files with your preferred tool
(Inkscape, ImageMagick, or design editor).
