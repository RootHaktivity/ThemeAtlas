# Security Policy

## Supported Versions

ThemeAtlas is currently maintained on the latest `main` branch.
Security fixes are prioritized for:

- Latest release
- Current `main` branch

## Reporting a Vulnerability

Please do not open public issues for undisclosed security vulnerabilities.

Report vulnerabilities by email:

- Address: sadistic.pentester@gmail.com
- Subject: ThemeAtlas Security Report

Please include:

- A clear description of the issue
- Steps to reproduce
- Impact assessment
- Affected environment (distro, desktop, Python version)
- Any proof-of-concept details

## Response Targets

- Initial acknowledgment: within 72 hours
- Status update: within 7 days
- Fix timeline: depends on severity and complexity

## Disclosure

After a fix is available, coordinated disclosure is encouraged.
If a CVE is needed, we will work with the reporter to provide details.
